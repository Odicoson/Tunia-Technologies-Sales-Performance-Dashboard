## Overview

Tunia Technologies is a Pan-African, tech-driven company founded in 2019, operating across three core verticals: **Marketplace**, **Logistics**, and **Payment Services**. This project involved designing and building a fully interactive sales performance dashboard using historical transactional data, enabling stakeholders to make faster, data-informed decisions across regions, product categories, and customer segments.

---

## Tools & Technologies

- **Microsoft Excel** – Power Query, Pivot Tables, Pivot Charts, Slicers, Conditional Formatting

## Dataset Fields

| Field | Description |
|---|---|
| Order Date | Date the customer placed the order |
| Ship Date | Date the product was shipped |
| Ship Mode | Shipping method selected by the customer |
| Customer Name | Name of the customer |
| Segment | Customer-type classification (Consumer, Corporate, Home Office) |
| Region | Geographical region (East, West, North, South) |
| Category | Broad product classification |
| Sub-Category | Specific product sub-classification |
| Product Name | Name of the purchased product |
| Sales | Revenue from the transaction |
| Profit | Net gain from the transaction |
| Quantity | Number of units sold |
| Discount | Discount applied to the purchase |

## Key Insights

- **Q4 is peak season** — Revenue consistently surges from October through December, with November being the strongest month across all years, peaking at 352K.
- **Consumer segment dominates profitability** — At 47% of total profit, Consumer customers are the most valuable segment to retain and reward.
- **Binders receive disproportionate discounts** — At $567.00, Binder discounts are more than 4x any other sub-category, warranting a pricing strategy review.
- **Office Supplies dominates volume** — 37% category share suggests strong repeat-purchase behavior and should anchor cross-sell strategies.

## How to Use

1. Download or clone this repository.
2. Open the `[Tunia Technologies – Sales & Performance Dashboard.xlsx](https://github.com/user-attachments/files/26778910/Tunia.Technologies.Sales.Performance.Dashboard.xlsx)
` file in **Microsoft Excel** (2016 or later recommended).
3. Use the slicers on the left panel to filter by Region, Ship Mode, or Year.
4. All charts and KPI cards update dynamically based on your selections.
